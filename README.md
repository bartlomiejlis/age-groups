# age-groups
Simple script which takes age of a person as an input and prints coresponding age group as an output.

The age groups are:
- Child: 0 to 11
- Teen: 12 to 17
- Adult: 18+

Live demo [_here_](https://replit.com/@BartlomiejLis/age-groups).
